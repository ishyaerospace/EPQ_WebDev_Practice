# JS Assignment operators
assignement operators assign value to variables

x = 10, y = 5
-----------------------------------
|operators|Example|Same As|Result |
-----------------------------------
| =       | x = y  | x = y | x = 5|
| +=      | x += y | x = x + y|x = 15|
| -=	  | x -= y |x = x - y |x = 5|
| *=	  | x *= y |x = x * y |x = 50|
| **=	  |x **= y |x = x ** y|	x = 100000|
|/=	x     |/= y	   |x = x / y |	x = 2|
|%=	x     |%= y	   |x = x % y |x = 0|
|:	      |x: 45   |size.x = 45|x = 45|
-----------------------------------
